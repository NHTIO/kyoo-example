This repository shows an example of using Kyoo

## How to run

To run this project, first create and populate a `.env` file. Then you can simply run:

```shell
node index.mjs
```

Or if you'd like to run just the producer or consumer, you can run:

```shell
node producer.mjs
```

and

```shell
node consumer.mjs
```
